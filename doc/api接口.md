# api功能说明文档

## 一、系统管理

### (一)、登陆

- 功能描述: 登陆系统

- api地址: /common/login

- 请求类型: POST

- 请求头:
```
无
```

- 请求url参数
```
无
```

- 请求体
```
{
  "loginName":"developer",							--登录名
  "loginPwd":"1"								--登陆密码
}
```

- 功能描述: 退出系统

- 请求类型: POST

- 请求头:
```
无
```

- 请求url参数
```
无
```

- 请求体
```
{
  "loginName":"developer",							--登录名
  "loginPwd":"1"								--登陆密码
}
```
